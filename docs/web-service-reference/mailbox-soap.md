# <a name="mailbox-soap"></a>Buzón de correo (SOAP)

El elemento de **buzón de correo** contiene la dirección de correo electrónico del usuario para ser detectado. 
  
```XML
<Mailbox/>
```

**string**

## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Element**|**Descripción**|
|:-----|:-----|
|[Usuario (SOAP)](user-soap.md) <br/> |Representa la identidad de un único usuario.  <br/> |
   
## <a name="text-value"></a>Valor de texto

El valor de texto del elemento de **buzón de correo** es la dirección de correo electrónico del usuario para ser detectado.
