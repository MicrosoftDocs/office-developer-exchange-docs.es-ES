# <a name="getfederationinformationrequest-soap"></a>GetFederationInformationRequest (SOAP)

El elemento **GetFederationInformationRequest** representa los parámetros de una llamada a la operación de la [operación de GetFederationInformation (SOAP)](getfederationinformation-operation-soap.md).
  
```XML
<GetFederationInformationRequest>
   <Domain/>
</GetFederationInformationRequest>
```

**GetFederationInformationRequest**
