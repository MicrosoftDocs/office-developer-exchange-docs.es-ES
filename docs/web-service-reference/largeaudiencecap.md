# <a name="largeaudiencecap"></a>LargeAudienceCap

El elemento **LargeAudienceCap** especifica el número máximo de destinatarios de un mensaje de correo electrónico. 
  
```XML
<LargeAudienceCap></LargeAudienceCap>
```

 **número entero**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

[MailTipsConfiguration (MailTipsServiceConfiguration)](mailtipsconfiguration-mailtipsserviceconfiguration.md)
  
## <a name="text-value"></a>Valor de texto

El valor de texto del elemento **LargeAudienceCap** es el número máximo de destinatarios que pueden incluirse en un mensaje de correo electrónico. 
  
## <a name="remarks"></a>Comentarios

Este elemento se introdujo en Exchange Server 2013.
  
El esquema que describe este elemento se encuentra en el directorio virtual IIS que hospeda los servicios Web Exchange.
