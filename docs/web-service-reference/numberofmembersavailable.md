# <a name="numberofmembersavailable"></a>NumberOfMembersAvailable

El elemento **NumberOfMembersAvailable** representa el número de miembros de la lista de distribución que están disponibles para una hora de reunión sugerida. Este elemento representa a los miembros para los que el estado es **gratuita**.
