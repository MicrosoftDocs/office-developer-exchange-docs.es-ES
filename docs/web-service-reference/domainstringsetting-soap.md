# <a name="domainstringsetting-soap"></a>DomainStringSetting (SOAP)

El elemento **DomainStringSetting** representa una configuración de dominio, el valor de los cuales es de tipo string. 
  
```XML
<DomainStringSetting>
   <Name/>
   <Value/>
</DomainStringSetting>
```

 **DomainStringSetting**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

|**Element**|**Descripción**|
|:-----|:-----|
|[Nombre (SOAP)](name-soap.md) <br/> |Representa el nombre de una opción de configuración.  <br/> |
|[Valor (SOAP)](value-soap.md) <br/> |Representa un valor de configuración de usuario.  <br/> |

Ninguno.
