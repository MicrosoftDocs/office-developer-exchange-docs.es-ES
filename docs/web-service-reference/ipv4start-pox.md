# <a name="ipv4start-pox"></a>IPv4Start (POX)

El elemento **IPv4Start** identifica el inicio de un intervalo de IP versión 4 (IPv4) las direcciones que se usan para identificar un equipo en una red.

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Element**|**Descripción**|
|:-----|:-----|

El valor de texto representa una dirección IPv4.
