# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

El elemento **AlternateMailboxes** representa una colección de buzones de correo alternativas. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
