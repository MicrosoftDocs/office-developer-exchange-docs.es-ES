# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

El elemento **AlternateMailboxes** representa una colección de buzones de correo alternativos. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```
