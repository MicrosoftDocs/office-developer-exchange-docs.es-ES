# <a name="votingresponse"></a>VotingResponse

El elemento **VotingResponse** especifica la votación enviada. Este elemento sólo está presente en las respuestas a mensajes de solicitud de voto, no en las respuestas a las aprobaciones. 
  
```XML
<VotingResponse />
```

 **string**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

[VotingInformation](votinginformation.md)
  
## <a name="text-value"></a>Valor de texto

El valor de texto del elemento **VotingResponse** es la votación enviada. 
  
## <a name="remarks"></a>Notas

Este elemento se incorporó en Exchange Server 2013 Service Pack 1 (SP1).
  
El esquema que describe este elemento se encuentra en el directorio virtual IIS que hospeda los servicios Web Exchange.
