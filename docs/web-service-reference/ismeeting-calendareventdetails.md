# <a name="ismeeting-calendareventdetails"></a>IsMeeting (CalendarEventDetails)

El elemento **IsMeeting** indica si el evento de calendario es una reunión o una cita.

## <a name="text-value"></a>Valor de texto

Si este elemento se devuelve en la respuesta, es necesario un valor de texto. Este elemento es necesario si se usa el elemento [CalendarEventDetails](calendareventdetails.md) . 
  
## <a name="remarks"></a>Comentarios

La diferencia entre una reunión y una cita es que una reunión es un elemento de calendario que incluye a los asistentes; una cita es un elemento de calendario que no incluye a los asistentes.
  
El esquema que describe este elemento se encuentra en el directorio virtual de EWS del equipo que está ejecutando MicrosoftExchange Server 2007 que tenga instalado el rol de servidor de acceso de cliente.
