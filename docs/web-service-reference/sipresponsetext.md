# <a name="sipresponsetext"></a>SIPResponseText

El elemento **SIPResponseText** especifica el texto de respuesta SIP. 
  
```xml
<SIPResponseText/>
```

 **string**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Element**|**Descripción**|
|:-----|:-----|
|[PhoneCallInformation](phonecallinformation.md) <br/> |Especifica la información de estado para una llamada de teléfono.  <br/> |
   
## <a name="text-value"></a>Valor de texto

El texto de respuesta SIP.
  
## <a name="remarks"></a>Comentarios

El esquema que describe este elemento se encuentra en el directorio /ews/ del equipo que ejecuta Exchange Server con la función de servidor de acceso de cliente instalada.
