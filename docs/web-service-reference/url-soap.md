# <a name="url-soap"></a>Dirección URL (SOAP)

El elemento de la **dirección Url** contiene la dirección URL de un cliente Web. 
  
```XML
<Url/>
```

 **string**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Element**|**Descripción**|
|:-----|:-----|
|[WebClientUrl (SOAP)](webclienturl-soap.md) <br/> |Representa la dirección URL de un cliente Web de Exchange.  <br/> |
   
## <a name="text-value"></a>Valor de texto

El valor de texto es la dirección URL de un cliente Web.
