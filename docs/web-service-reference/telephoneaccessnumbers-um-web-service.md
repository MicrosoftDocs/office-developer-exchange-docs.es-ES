# <a name="telephoneaccessnumbers-um-web-service"></a>TelephoneAccessNumbers (servicio web de mensajería unificada)

El elemento **TelephoneAccessNumbers** contiene un valor que es una lista de números de teléfono que los usuarios pueden usar para tener acceso a mensajería unificada, tal como lo devuelve en respuesta a una solicitud de [operación GetUMProperties (servicio web de mensajería unificada)](getumproperties-operation-um-web-service.md) . 
  
[GetUMPropertiesResponse (servicio web de mensajería unificada)](getumpropertiesresponse-um-web-service.md)
  
[TelephoneAccessNumbers (servicio web de mensajería unificada)](telephoneaccessnumbers-um-web-service.md)

```xml
<TelephoneAccessNumbers/>
```

 **string**
