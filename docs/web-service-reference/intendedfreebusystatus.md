# <a name="intendedfreebusystatus"></a>IntendedFreeBusyStatus

El elemento **IntendedFreeBusyStatus** representa el estado deseado para el elemento de calendario que está asociado con la convocatoria de reunión. 
  
```xml
<IntendedFreeBusyStatus/>
```

 **LegacyFreeBusyType**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Element**|**Descripción**|
|:-----|:-----|
|[MeetingRequest](meetingrequest.md) <br/> |Representa una convocatoria de reunión en el almacén de Exchange.  <br/> |
   
## <a name="text-value"></a>Valor de texto

Se requiere un valor de texto. Los siguientes son los valores posibles para este elemento:
  
- Gratuito
    
- Provisional
    
- Ocupado
    
- FUERA DE LA OFICINA
    
- NoData
    
## <a name="remarks"></a>Comentarios

El esquema que describe este elemento se encuentra en el directorio virtual de EWS del equipo que ejecuta Microsoft Exchange Server 2007 que tenga instalado el rol de servidor de acceso de cliente.
