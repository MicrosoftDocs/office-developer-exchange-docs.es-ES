# <a name="autodiscoverrequest-soap"></a>AutodiscoverRequest (SOAP)

El elemento **AutodiscoverRequest** representa el elemento base para todas las solicitudes de detección automática. 
  
```XML
<AutodiscoverRequest/>
```
