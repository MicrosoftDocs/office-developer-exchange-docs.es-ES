# <a name="members-memberlisttype"></a>Miembros (MemberListType)

El elemento **Members** proporciona la lista de miembros para una lista de distribución. 
  
```xml
<Members>
   <Member/>
</Members>
```

**MemberListType**

## <a name="attributes-and-elements"></a>Atributos y elementos

En las siguientes secciones se describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguna.
  
### <a name="child-elements"></a>Elementos secundarios

|**Elemento**|**Descripción**|
|:-----|:-----|
|[Miembro](member-ex15websvcsotherref.md) <br/> |Proporciona un identificador para una dirección de correo electrónico completamente resuelta y el estado de esa dirección en el servidor. Este elemento es opcional.  <br/> |
   
### <a name="parent-elements"></a>Elementos principales

|**Elemento**|**Descripción**|
|:-----|:-----|
|[DistributionList](distributionlist.md) <br/> |Representa una lista de distribución.  <br/> |
   
## <a name="remarks"></a>Comentarios

El esquema que describe este elemento se encuentra en el directorio virtual IIS que hospeda los servicios Web Exchange.
