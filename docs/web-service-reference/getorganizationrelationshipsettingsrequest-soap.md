# <a name="getorganizationrelationshipsettingsrequest-soap"></a>GetOrganizationRelationshipSettingsRequest (SOAP)

El elemento **GetOrganizationRelationshipSettingsRequest** representa los parámetros de una llamada a la operación de la [operación de GetOrganizationRelationshipSettings (SOAP)](getorganizationrelationshipsettings-operation-soap.md) . El elemento **GetOrganizationRelationshipSettingsRequest** es sólo para uso interno. Este elemento no se usa en los clientes. 
  
```XML
<GetOrganizationRelationshipSettingsRequest>
   <Domains/>
</GetOrganizationRelationshipSettingsRequest>
```
