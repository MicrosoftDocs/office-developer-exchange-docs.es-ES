# <a name="displaycc"></a>DisplayCc

El elemento **DisplayCc** representa la cadena para mostrar que se usa para el contenido del cuadro Cc. Ésta es la cadena concatenada de todos los nombres de presentación de los destinatarios de Cc. 
  
```xml
<DisplayCc/>
```

 **String**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.

## <a name="text-value"></a>Valor de texto

El valor de texto representa la cadena para mostrar que se usa para el contenido del cuadro Cc.
  
## <a name="remarks"></a>Comentarios

El esquema que describe este elemento se encuentra en el directorio virtual de EWS del equipo que ejecuta Microsoft Exchange Server 2007 que tenga instalado el rol de servidor de acceso de cliente.
