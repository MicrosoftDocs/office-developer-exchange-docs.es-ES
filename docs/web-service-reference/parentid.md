# <a name="parentid"></a>ParentId

El elemento **ParentId** especifica el identificador del elemento primario en una vista previa de la búsqueda. 
  
```XML
<ParentId Id="" ChangeKey=""/>
```

**ItemIdType**

## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

|**Attribute**|**Descripción**|
|:-----|:-----|
|Id  <br/> |El valor de texto del atributo **Id** es el identificador del elemento primario.  <br/> |
|ChangeKey  <br/> |El valor de texto del atributo **ChangeKey** es la clave de cambio del elemento primario.  <br/> |
   
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

[SearchPreviewItem](searchpreviewitem.md)
  
## <a name="remarks"></a>Comentarios

Este elemento se introdujo en Exchange Server 2013.
  
El esquema que describe este elemento se encuentra en el directorio virtual IIS que hospeda los servicios Web Exchange.
