# <a name="contactstring"></a>ContactString

El elemento **ContactString** especifica el nombre para mostrar de un contacto. 
  
```XML
<ContactString></ContactString>
```

 **string**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Element**|**Descripción**|
|:-----|:-----|
|[Contacto (ContactType)](contact-contacttype.md) <br/> |Especifica un contacto en el almacén de contactos unificados.  <br/> |
   
## <a name="text-value"></a>Valor de texto

El valor de texto del elemento **ContactString** es el valor de tipo string que especifica el nombre para mostrar. 
  
## <a name="remarks"></a>Comentarios

Este elemento se introdujo en Exchange Server 2013.
  
El esquema que describe este elemento se encuentra en el directorio virtual IIS que hospeda los servicios Web Exchange.
