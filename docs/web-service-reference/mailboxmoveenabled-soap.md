# <a name="mailboxmoveenabled-soap"></a>MailboxMoveEnabled (SOAP)

El elemento **MailBoxMoveEnabled** representa la marca **MailboxMoveEnabled()** . El elemento **MailBoxMoveEnabled** es sólo para uso interno. Este elemento no se usa en los clientes. 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**

## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
