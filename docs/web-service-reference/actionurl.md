# <a name="actionurl"></a>ActionUrl

El elemento **ActionUrl** identifica la dirección URL que debería ir el usuario, con el fin de corregir un problema indicado por el elemento [AppStatus](appstatus-ex15websvcsotherref.md) . 
  
```XML
<ActionUrl/>
```

 **string**
## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

[Metadatos](metadata-ex15websvcsotherref.md)
  
## <a name="text-value"></a>Valor de texto

El valor de texto del elemento **ActionUrl** identifica la dirección URL que debería ir el usuario, con el fin de corregir un problema indicado por el elemento **AppStatus** . 
  
## <a name="remarks"></a>Notas

Este elemento se incorporó en Exchange Server 2013 Service Pack 1 (SP1).
  
El esquema que describe este elemento se encuentra en el directorio virtual IIS que hospeda los servicios Web Exchange.
