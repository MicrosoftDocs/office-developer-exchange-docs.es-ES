## <a name="to-use-the-ews-managed-api-to-communicate-with-ews"></a><span data-ttu-id="cb60b-146">Para usar la API administrada de EWS para comunicarse con EWS</span><span class="sxs-lookup"><span data-stu-id="cb60b-146">To use the EWS Managed API to communicate with EWS</span></span>

1. <span data-ttu-id="cb60b-147">Crear una instancia de la clase **ExchangeService** .</span><span class="sxs-lookup"><span data-stu-id="cb60b-147">Instantiate the **ExchangeService** class.</span></span> 
    
   ```csharp
    ExchangeService service = new ExchangeService();
   ```

   > [!NOTE]
   > <span data-ttu-id="cb60b-148">Creación de instancias de **ExchangeService** con un constructor vacío, se creará una instancia que está enlazada a la última versión conocida de Exchange.</span><span class="sxs-lookup"><span data-stu-id="cb60b-148">Instantiating **ExchangeService** with an empty constructor will create an instance that is bound to the latest known version of Exchange.</span></span> <span data-ttu-id="cb60b-149">Como alternativa, puede dirigir una versión específica de Exchange mediante la especificación de versión como un parámetro.</span><span class="sxs-lookup"><span data-stu-id="cb60b-149">Alternatively, you can target a specific version of Exchange by specifying version as a parameter.</span></span> `ExchangeService service = new ExchangeService(ExchangeVersion.Exchange2007_SP1);`
  
2. <span data-ttu-id="cb60b-150">Establecer las credenciales del usuario que envía las solicitudes al servidor de Exchange.</span><span class="sxs-lookup"><span data-stu-id="cb60b-150">Set the credentials of the user who sends requests to the Exchange server.</span></span> <span data-ttu-id="cb60b-151">Si desea conectarse a EWS desde un equipo que ha iniciado sesión en el dominio, con las credenciales del usuario autenticado, establezca la propiedad **UseDefaultCredentials** en el objeto **ExchangeService** en **true**.</span><span class="sxs-lookup"><span data-stu-id="cb60b-151">If you want to connect to EWS from a computer that is logged on to the domain, using the credentials of the authenticated user, set the **UseDefaultCredentials** property on the **ExchangeService** object to **true**.</span></span>
    
   ```cs
    // Connect by using the default credentials of the authenticated user.
    service.UseDefaultCredentials = true;
   ```

   <span data-ttu-id="cb60b-152">Si no desea conectarse mediante el uso de las credenciales del usuario de forma predeterminada, establezca la propiedad de **credenciales** en el objeto **ExchangeService** para especificar explícitamente las credenciales de un usuario diferente.</span><span class="sxs-lookup"><span data-stu-id="cb60b-152">If you do not want to connect by using the default user credentials, set the **Credentials** property on the **ExchangeService** object to explicitly specify the credentials of a different user.</span></span> <span data-ttu-id="cb60b-153">Si usa Exchange Online o Exchange Online como parte de Office 365, que usará la autenticación básica, con sólo un nombre de usuario y una contraseña.</span><span class="sxs-lookup"><span data-stu-id="cb60b-153">If you are using Exchange Online or Exchange Online as part of Office 365, you'll use basic authentication, with just a user name and password.</span></span> <span data-ttu-id="cb60b-154">Un nombre de dominio se requiere para la autenticación NTLM.</span><span class="sxs-lookup"><span data-stu-id="cb60b-154">A domain name is required for NTLM authentication.</span></span> 
    
   ```cs

   ```

   <span data-ttu-id="cb60b-155">También puede especificar las credenciales del usuario mediante el uso de nombre de dominio del usuario y la contraseña.</span><span class="sxs-lookup"><span data-stu-id="cb60b-155">You can also specify the credentials of the user by using the user's domain name and password.</span></span>


   > [!NOTE]
   > <span data-ttu-id="cb60b-156">Si se establece la propiedad **UseDefaultCredentials** en **true**, se omite el valor de la propiedad de **credenciales** .</span><span class="sxs-lookup"><span data-stu-id="cb60b-156">If the **UseDefaultCredentials** property is set to **true**, the value of the **Credentials** property is ignored.</span></span>
