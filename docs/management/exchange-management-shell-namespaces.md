# <a name="exchange-management-shell-namespaces"></a>Espacios de nombres del shell de administración de Exchange

Busque información sobre los espacios de nombres para el shell de administración de Exchange en Exchange.
  
**Se aplica a:** Exchange Online | Exchange Server 2013 | Office 365
  
Aplicaciones Shell de administración de Exchange para Exchange Online, Exchange online como parte de Office 365 o una versión de Exchange que empiece con Exchange 2013 interactúe con el entorno de Windows PowerShell a través de los tipos expuestos por el espacio de nombres **System. Management. Automation** . Los cmdlets del shell de administración de Exchange usan tipos de una serie de bibliotecas de Exchange. En esta sección se proporciona información de referencia sobre los tipos y las interfaces de las bibliotecas de Exchange que usan los cmdlets del shell de administración de Exchange.
